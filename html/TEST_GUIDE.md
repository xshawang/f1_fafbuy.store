## 快速开始

### 1. 启动本地服务器

**方法 A - 使用 Python（推荐）**
```bash

python -m http.server 8000
```

**方法 B - 使用 Node.js**
```bash
cd "d:\work\project\java\wuhan\f1\html"

```

**方法 C - 使用 VS Code**
- 安装 "Live Server" 扩展
- 右键 `checkout-demo.html`
- 选择 "Open with Live Server"

打开浏览器访问：


## 常见问题

### Q1: 页面无法加载 JSON 文件
**错误：** CORS error 或 Failed to load resource

**解决方案：**
- 必须使用 HTTP 服务器，不能直接打开 HTML 文件
- 确保服务器正在运行
- 检查文件路径是否正确

### Q2: 语言切换不生效
**检查：**
- 打开浏览器控制台（F12）
- 查看是否有 JavaScript 错误
- 确认 JSON 文件格式正确

### Q3: 国家下拉框为空
**检查：**
- 确认 `js/countries.json` 文件存在
- 查看控制台是否有加载错误
- 刷新页面重试

### Q4: 信用卡验证不工作
**检查：**
- 确认 `js/credit-card.js` 已加载
- 查看控制台是否有错误
- 确保输入框 ID 正确（cardNumber, cardExpiry, cardCvv, cardName）
