**簡要描述：** 

- 用戶現存於平台的總餘額

**請求URL：** 
- ` /getpoints `
  
**請求方式：**
- POST 

**參數：** 

|參數名|參數|類型|說明|
|:-----  |:-----|:-----|-----                           |
|商戶ID|uid|string(5)|商戶ID號|
|時間戳|timestamp|int|GMT時間戳|
|數據簽名|sign|string(32)|32位大寫MD5簽名值  |

 **返回示例**

#### 單一幣別的情況下（TWD）
``` 
  {
    "status": 10000,
    "result": {
        "points":"100.00",
    }.
    "sign":"xxx"
  }
```

  **返回參數中答复資料說明** 

|參數名|參數|類型|說明|
|:-----  |:-----|:-----|-----                           |
|狀態代碼|status |int   |更多返回錯誤代碼請看狀態碼表  |
|答复資料|result |object   |當狀態碼為「成功」時返回JSON字串  |
|數據簽名|sign |string   |32位大寫MD5簽名值  |

 **返回參數中JSON字串說明** 

|參數名|參數|類型|說明|
|:-----  |:-----|:-----|-----                           |
|金額|points |float    |交易成功且確實收到款的金額，範圍小數點後兩位|

#### 含有多幣別的情況下
``` 
  {
    "status": 10000,
    "result":{
        "0":{
            "currencyID":36,
            "points":"5233.00"
        },
        "1":{
            "currencyID":42,
            "points":"0.00"
        },
        "2":{
            "currencyID":51,
            "points":"120198.72"
        },
        "3":{
            "currencyID":88,
            "points":"16298401.25"
        }
    },
    "sign":"xxx"
  }
```
  **返回参数中答复资料说明** 

|参数名|参数|类型|说明|
|:-----  |:-----|:-----|-----                           |
|状态代码|status |int   |更多返回错误代码请看状态码表  |
|答复资料|result |object   |当状态码为「成功」时返回JSON字串  |
|数据签名|sign |string   |32位大写MD5签名值  |

 **返回参数中JSON字串说明** 

|参数名|参数|类型|说明|
|:-----  |:-----|:-----|-----                           |
|币别|currencyID|int|参数值见「币别」|
|金额|points |float    |交易成功且确实收到款的金额，范围小数点后两位|

 **备注** 
- 返回参数中的答复资料是以有序物件组成
- 系统限制每 5 秒可查询一次
